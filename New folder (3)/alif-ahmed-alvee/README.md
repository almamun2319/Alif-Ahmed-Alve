# আলিফ আহমেদ আল্ভী - ফটো ও ভিডিও গ্যালারি

একটি সুন্দর এবং আধুনিক ফটো ও ভিডিও গ্যালারি ওয়েবসাইট।

## বৈশিষ্ট্য

✨ **সুন্দর ডিজাইন** - আধুনিক এবং প্রতিক্রিয়াশীল ইন্টারফেস  
📱 **মোবাইল বান্ধব** - সব ডিভাইসে নিখুঁত প্রদর্শন  
🖼️ **ছবি গ্যালারি** - সহজে ছবি প্রদর্শন করুন  
🎥 **ভিডিও সমর্থন** - YouTube ভিডিও এম্বেড করুন  
🔍 **ফিল্টার বৈশিষ্ট্য** - সব, ছবি, বা ভিডিও ফিল্টার করুন  
💾 **বিনামূল্যে হোস্টিং** - Vercel, GitHub Pages এ চিরকাল লাইভ

## স্থানীয়ভাবে চালানো

### প্রয়োজনীয়তা
- আধুনিক ওয়েব ব্রাউজার

### চালান
```bash
# Python 3 ব্যবহার করে সার্ভার চালান
python -m http.server 8000

# তারপর ব্রাউজারে খুলুন:
# http://localhost:8000
```

## অনলাইনে হোস্ট করা - বিনামূল্যে

### অপশন ১: GitHub Pages (সবচেয়ে সহজ)

1. GitHub এ একটি অ্যাকাউন্ট তৈরি করুন (যদি না থাকে)
2. এই ফোল্ডারটিকে একটি রিপোজিটরিতে আপলোড করুন
3. `main` ব্রাঞ্চে পুশ করুন
4. GitHub Actions স্বয়ংক্রিয়ভাবে সাইট ডিপ্লয় করবে
5. Repository settings → "Pages" এ গিয়ে ব্রাঞ্চ হিসেবে `gh-pages` নির্বাচন করুন
6. **Done!** আপনার সাইট লাইভ হবে: `https://yourusername.github.io/alif-ahmed-alvee`

> এই প্রোজেক্টে `.github/workflows/deploy.yml` ফাইল আছে, যা `main` ব্রাঞ্চে পুশ করলে স্বয়ংক্রিয়ভাবে GitHub Pages এ ডিপ্লয় করে।

### অপশন ২: Vercel (দ্রুত এবং সহজ)

1. [Vercel.com](https://vercel.com) এ যান
2. GitHub দিয়ে সাইন ইন করুন
3. আপনার রিপোজিটরি ইমপোর্ট করুন
4. **Deploy** বাটনে ক্লিক করুন
5. আপনার সাইট তাৎক্ষণিকভাবে লাইভ হবে

### অপশন ৩: Netlify

1. [Netlify.com](https://netlify.com) এ যান
2. "Add new site" → "Import an existing project" নির্বাচন করুন
3. আপনার GitHub রিপোজিটরি সংযুক্ত করুন
4. ডিপ্লয় করুন

### অপশন ৪: Firebase Hosting (Google)

1. [Firebase Console](https://console.firebase.google.com) এ যান
2. একটি প্রোজেক্ট তৈরি করুন
3. Hosting সেকশনে যান
4. `firebase init` রান করুন
5. `firebase deploy` রান করুন

## ছবি এবং ভিডিও যোগ করা

### script.js ফাইল খুলুন এবং `galleryItems` অ্যারেতে নতুন আইটেম যোগ করুন:

```javascript
// ছবি যোগ করার উদাহরণ:
{
    id: 9,
    type: 'photo',
    src: 'আপনার_ছবির_লিঙ্ক.jpg',
    caption: 'ছবির বর্ণনা'
}

// ভিডিও যোগ করার উদাহরণ (YouTube):
{
    id: 10,
    type: 'video',
    src: 'https://www.youtube.com/embed/ভিডিও_আইডি',
    caption: 'ভিডিওর বর্ণনা',
    thumbnail: 'https://img.youtube.com/vi/ভিডিও_আইডি/maxresdefault.jpg'
}
```

### ছবি আপলোড করার স্থান:
1. **Imgur** - ছবি শেয়ারিং সার্ভিস (বিনামূল্যে)
2. **imgur.cloud** - সরাসরি ছবি হোস্ট করুন
3. **GitHub** - আপনার রিপোজিটরিতে `images` ফোল্ডার তৈরি করুন

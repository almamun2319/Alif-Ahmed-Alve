## ধাপ ২: আপনার কম্পিউটারে পরীক্ষা করুন

### Windows এ:

1. **Command Prompt খুলুন**
   - Windows + R চাপুন
   - `cmd` টাইপ করুন এবং Enter চাপুন

2. **এই কমান্ড চালান:**
   ```bash
   cd "d:\New folder (3)\alif-ahmed-alvee"
   python -m http.server 8000
   ```

3. **ব্রাউজারে খুলুন:**
   - `http://localhost:8000` লিংকে যান

### Mac/Linux এ:

```bash
cd ~/path/to/alif-ahmed-alvee
python3 -m http.server 8000
```

## ধাপ ৩: ছবি এবং ভিডিও যোগ করা

### সবচেয়ে সহজ উপায় - ছবি যোগ করুন Imgur থেকে:

1. **Imgur.com এ যান** (বিনামূল্যে অ্যাকাউন্ট তৈরি করুন)
2. **ছবি আপলোড করুন** এবং Share লিঙ্ক কপি করুন
3. **script.js ফাইল খুলুন** এবং এই কোড যোগ করুন:

```javascript
{
    id: 9,
    type: 'photo',
    src: 'আপনার_imgur_লিঙ্ক',
    caption: 'আপনার ছবির নাম'
}
```

### YouTube ভিডিও যোগ করুন:

1. **YouTube ভিডিও খুলুন**
2. **Share → Embed করুন** এবং লিঙ্ক কপি করুন
3. **script.js ফাইলে যোগ করুন:**

```javascript
{
    id: 10,
    type: 'video',
    src: 'https://www.youtube.com/embed/ভিডিও_আইডি',
    caption: 'ভিডিওর নাম',
    thumbnail: 'https://img.youtube.com/vi/ভিডিও_আইডি/maxresdefault.jpg'
}
```

## ধাপ ৪: অনলাইনে লাইভ করা (বিনামূল্যে)

### বিকল্প A: GitHub Pages (সবচেয়ে সহজ) ⭐

#### উইন্ডোজে:

1. **GitHub.com এ যান এবং অ্যাকাউন্ট তৈরি করুন**
   - https://github.com
   - ইমেল এবং পাসওয়ার্ড দিয়ে সাইন আপ করুন

2. **নতুন রিপোজিটরি তৈরি করুন:**
   - "+" আইকন → "New repository"
   - নাম: `alif-ahmed-alvee`
   - "Public" নির্বাচন করুন
   - "Create repository" ক্লিক করুন

3. **ফাইলগুলি আপলোড করুন:**
   - "Add file" → "Upload files"
   - আপনার সব ফাইল নির্বাচন করুন এবং আপলোড করুন

4. **Pages সক্ষম করুন:**
   - Repository settings খুলুন
   - বাম পাশে "Pages" খুঁজুন
   - Branch: `main` নির্বাচন করুন
   - ফোল্ডার: `/ (root)` নির্বাচন করুন
   - "Save" ক্লিক করুন

5. **সাইট লাইভ হয়ে যাবে!**
   - URL: `https://yourusername.github.io/alif-ahmed-alvee`
   - এটি সর্বদা লাইভ থাকবে! 🎉

## সমস্যা সমাধান

### সাইট খোলা যাচ্ছে না?

- সার্ভার চালু আছে কিনা চেক করুন: `http://localhost:8000`
- যদি সার্ভার বন্ধ থাকে, উপরের কমান্ড আবার চালান

### ছবি দেখা যাচ্ছে না?

- ছবির লিঙ্ক সঠিক কিনা চেক করুন
- ব্রাউজারের ডেভেলপার টুলস খুলুন (F12)
- Console ট্যাবে এরর দেখুন

### GitHub Pages এ আপডেট হচ্ছে না?

- Repository settings → Pages এ যান
- Branch সঠিক আছে কিনা চেক করুন
- 5 মিনিট অপেক্ষা করুন (সাধারণত তাৎক্ষণিক হয়)

## উন্নত: নিজের ডোমেইন ব্যবহার করুন

### বিনামূল্যে ডোমেইন:

- **Freenom.com** - বিনামূল্যে .tk ডোমেইন
- **Nameserver** - ফ্রি ডাইনামিক ডোমেইন

### ডোমেইন সংযুক্ত করুন:

1. আপনার ডোমেইন প্রদানকারীতে যান
2. **DNS Settings** খুলুন
3. এই রেকর্ড যোগ করুন (GitHub Pages এর জন্য):

```
Type: CNAME
Name: www
Value: yourusername.github.io
```

4. **Save** ক্লিক করুন এবং অপেক্ষা করুন
